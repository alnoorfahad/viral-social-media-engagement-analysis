
## Objectives
- Identify factors that contribute to viral posts (platform, content type, hashtags, region).
- Visualise engagement patterns and relationships (views vs engagement, etc).
- Provide actionable insights for content optimization.

## Tech Stack
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Google Colab / Jupyter Notebook

## How to run
1. Open `notebook/viral_social_media_analysis.ipynb` on GitHub or load it in Google Colab.
2. If running locally, install dependencies:
